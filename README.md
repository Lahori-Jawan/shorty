# shorty

A URL shortening Fullstack Vue App.

# Development

## clone

`git clone https://github.com/Lahori-Jawan/shorty.git .`

## For Docker

`docker-compouse up`

## For Local

```
npm install
npm run dev
```

# Functionality

- Auth
  - Login
  - Register
- Shorten a URL
- Buy a custom domain
- Activate custom domain
- Get URLs for a specific user
- Get User Profile
